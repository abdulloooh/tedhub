Chem1 Virtual Textbook download information

You have downloaded the file chem1vt.zip and have unzipped it, which yields the folder named chem1vt, whose structure is as follows:

chem1vt

webtext (the main menu virtualtextbook.html is located here)

webtut

README.html (the file you are now viewing)

1. Locate the chem1vt folder (it will likely be in your downloads folder). Move this folder and its contents to a convenient place on your hard drive.

2. Open the subdirectory webtext, and locate the file virtualtextbook.html, which is the main menu entry into the Virtual Textbook. Double-click to open it in your Web browser, and bookmark its location.

To locate virtualtextbook.html:

  • If you are using Windows, press the "Windows" key in the bottom row of your keyboard, or click on the Windows icon at the bottom left of your screen. Then click on "computer" in the right panel, and then on the drive (usually C) on which the chem1vt folder resides (possibly in a "downloads" folder.)
  • On a Macintosh system, chem1vt will probably be in your downloads folder. Use the finder to locate it, and drag it to a more convenient location.
  • On Linux, activate the menu and click on "computer", then the appropriate "user" folder and then "downloads".

Web Browser note

The Virtual Textbook is best used with a standards-compliant Web browser such as Chrome, Firefox, Opera, or Safari.

Microsoft Internet Explorer (even version 8) will not render some pages correctly.

In order to utilize all features, Javascript must be turned on. This can usually be done through the "security" panel within the browser's preferences control. On Windows, you may need to turn "protection" off.
